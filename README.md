## Introduction

This project demonstrates a simple Machine Learning model that predicts student marks based on the number of study hours. It was created to understand the fundamentals of predictive analytics and supervised learning using Python.

The project uses the Linear Regression algorithm to identify the relationship between study hours and exam marks. After training the model, predictions are generated and evaluated using standard performance metrics.

This project provides hands-on experience with the complete Machine Learning workflow, including data preparation, model training, prediction, evaluation, and visualization.



## Project Objectives

The main objectives of this project are:

* Understand the basics of Machine Learning
* Learn how Linear Regression works
* Build a predictive model using Python
* Evaluate model performance
* Visualize relationships between variables



## Features

✅ Dataset creation using Pandas

✅ Data splitting into training and testing sets

✅ Linear Regression model training

✅ Prediction of student marks

✅ Model performance evaluation

* Python
* Pandas

## Machine Learning Workflow

### 1. Dataset Creation

A sample dataset is created containing:

* Study Hours
  
* Student Marks

### 2. Data Preparation

The dataset is divided into:

* Features (Study Hours)
* Target Variable (Marks)

### 3. Train-Test Split

The data is split into training and testing sets to evaluate model performance.

### 4. Model Training

A Linear Regression model is trained using the training dataset.

### 5. Prediction

The trained model predicts marks based on study hours.

### 6. Model Evaluation

Performance is measured using:

* R² Score
* 
* Mean Squared Error (MSE)


### 7. Visualization

A regression line is plotted to show the relationship between study hours and marks.




## Sample Output


R2 Score: 0.95
MSE: 12.50


## Visualization Output

The project generates a graph showing:

* Data points representing study hours and marks
* A regression line representing the model's predictions

This visualization helps in understanding how marks tend to increase as study hours increase.
